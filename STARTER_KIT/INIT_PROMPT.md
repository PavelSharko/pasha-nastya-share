# Инструкция для Агента (INIT PROMPT)

**Привет, Агент!** Я передал тебе эту директорию в качестве основы для моей новой Базы Знаний (Второго Мозга) в формате "Hybrid LLM Wiki". Здесь уже заложены базовые правила и скиллы. 

Твоя задача сейчас — инициализировать эту рабочую среду под мои потребности. 

Пожалуйста, выполни следующие шаги строго по порядку:

### Шаг 1: Изучение правил
Внимательно прочитай файл `GEMINI.md` в корне этой директории. Там описаны твои ограничения (guardrails), базовые команды (`scan_raw`, работа с сессиями) и правила оформления заметок. Это твой главный конституционный документ.

### Шаг 2: Сканирование и Адаптация Скиллов
В папке `.agents/skills/` находятся готовые навыки (например, `session-manager`, `skill-creator`, `google-sheets-analyst` и другие). 
1. Просканируй их.
2. Подтверди, что ты видишь эти скиллы. С этого момента ты ДОЛЖЕН использовать их, особенно `session-manager` для сохранения истории и контекста.

### Шаг 3: Адаптация Доменов
Прямо сейчас в хранилище есть шаблонные папки: `DOMAIN_1_WORK`, `DOMAIN_2_PERSONAL`, `DOMAIN_3_HOBBY`.
Я хочу, чтобы ты помог мне настроить их. 
1. Задай мне 3-4 вопроса о том, чем я занимаюсь (моя профессия, хобби, какие данные я хочу здесь хранить).
2. На основе моих ответов переименуй эти папки (например, вместо `DOMAIN_1_WORK` сделай `ПРОГРАММИРОВАНИЕ` или `МАРКЕТИНГ`). 
3. Обнови файл `GEMINI.md`, заменив шаблонные `DOMAIN_*` на мои реальные новые папки, чтобы правила применялись к ним.

### Шаг 4: Финальный Рапорт
Как только мы закончим настройку папок, выдай мне **Финальный Отчет Инициализации**. 
В отчете укажи:
- Какие скиллы ты успешно подключил и можешь использовать.
- Какие базовые команды (например, `scan_raw`) ты теперь понимаешь.
- Какая структура папок у нас получилась.
- Предложи мне первое действие (например, скинуть первую сырую заметку в папку `RAW/`).

**Жду твоих вопросов по Шагу 3, чтобы мы могли настроить мои домены!**
