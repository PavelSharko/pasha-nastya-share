# Промпт для создания промпта (Comet)

Этот промпт используется для того, чтобы ChatGPT составил качественную инструкцию для умного браузера Comet.

---

**Текст промпта:**
"Окей, я хочу, чтобы ты составил мне промпт для умного браузера Comet, который умеет управлять браузером на основании моих инструкций. 

Задача: дай мне готовый промпт для поиска актуальных тем за последнюю неделю. Результат поиска сохрани в папку RAW."
