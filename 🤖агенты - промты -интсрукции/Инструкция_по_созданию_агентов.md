# Инструкция по созданию агентов

Этот документ описывает логику создания автономных агентов на базе Python-скриптов, которые могут взаимодействовать с браузером и внешними инструментами.

## Структура агента (на примере Aget_for_erank)
Агент должен состоять из:
1. `setup_session.py`: Настройка окружения и сессии.
2. `worker.py`: Основная логика работы (скрейпинг, скроллинг, обработка данных).
3. `AGENT_INTERFACE.md`: Описание команд и параметров для взаимодействия.

## Принципы реализации
- **Не "тупой" поиск**: Агент не должен просто выдавать готовую информацию от ИИ. Он обязан использовать качественный поиск, скрейпинг и имитацию действий пользователя (скроллинг).
- **Использование сессий**: Для качественной работы можно использовать куки (cookies) сессий через специально открытый браузер.
- **Скрипты**: Вся работа должна быть автоматизирована через скрипты.

## План разработки (Draft)
1. Изучить задачу и параметры, передаваемые агенту.
2. Подготовить структуру папок.
3. Реализовать скрейпер с поддержкой сессий.
4. Протестировать на реальных данных (например, Etsy или Google Trends).
