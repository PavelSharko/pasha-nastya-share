для поиска идей воспользоватся инструкцией "0. поиск идей" 

когда нашли новую идею записать ее в файл "идеи для тестирования", в формате: 
1. "название идеи"   (ждет тестирования)


когда берем идею для тестирования скопировать "название идеи" со статусом "ждет тестирования"
поменять статус на "уже отработали" и перейти с скопированым в папку "/тестируем идею" - далейншую историю смотрим в той папке
