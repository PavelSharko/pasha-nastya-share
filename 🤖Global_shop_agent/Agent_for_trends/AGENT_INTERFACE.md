# Инструкция по запуску Trend Agent (API для ИИ-агентов)

Этот файл описывает, как другим ИИ-агентам вызывать скрипт `trend_worker.py` для автономного поиска ниш без вызова графического меню.

## Как запустить агента
Для автоматического запуска скрипт нужно вызвать с флагом `--status_run auto`.

### Доступные параметры:

| Флаг | Значение по умолчанию | Описание |
| :--- | :--- | :--- |
| `--status_run` | `manual` | Обязательно установить в `auto`, чтобы пропустить красивое меню (TUI). |
| `--geo` | `global` | Регион поиска для Google Trends. Например: `US` (США), `GB` (Британия), `EU` (Европа), или пусто для всего мира. |
| `--date` | `now 7-d` | Временной промежуток. Например: `now 1-d` (сутки), `now 7-d` (неделя), `today 1-m` (месяц). |
| `--category`| `0` | ID категории в Google Trends. `0` = все категории, `e` = Развлечения, `t` = Технологии. |

### Пример команды для запуска из другого агента:
```bash
python trend_worker.py --status_run auto --geo US --date "now 7-d" --category e
```

### Куда сохраняется результат?
Агент генерирует Markdown-отчет с вшитым JSON-блоком статусов.
Все статусы проверок изначально устанавливаются в `"no"`.
Результат всегда сохраняется в папку:
`🤖Global_shop_agent/база_данных_для_агентов/actual_topics/`
