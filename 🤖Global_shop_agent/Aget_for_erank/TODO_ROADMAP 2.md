# 🗺️ Roadmap: eRank Agent v2.0

Список задач по улучшению агента для глубокого анализа тегов.

## 🚀 Функционал
- [ ] **Полные отчеты**: Снимать ограничение в 50-100 строк в MD, сохранять все доступные теги (до 500-1000+).
- [ ] **Мульти-региональность**: Возможность выбрать сразу несколько стран (например, USA + EEA + GBR) и получить сводный отчет.
- [ ] **Графики спроса**: Собирать данные по месяцам (Monthly Search Trend), чтобы видеть сезонность.
- [ ] **Фоновый запуск**: Настройка расписания (Cron/Launchd), чтобы агент собирал данные ночью.

## 📊 Аналитика
- [ ] **Анализ по правилам**: Допилить автоматический вывод (например: "Тег хороший, низкая конкуренция").
- [ ] **Проверка лимитов**: Убедиться, что агент стабильно выкачивает 500+ строк без зависаний.

## 🛠 Техническое
- [ ] **Формат сохранения**: Оптимизировать структуру папок (например, `RAW/Keyword_Reports/Keyword_Name/`).
- [ ] **Умные тайм-ауты**: Улучшить логику ожидания для очень медленных страниц eRank.
